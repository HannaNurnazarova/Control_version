# Подсказка по командной строке


Команда смены дериктории 

'''
cd с:\folder_name 
'''

Команда отображения текущей дериктории

'''
pwd
''' 

Листинг текущей директории
Windows:

'''
dir
'''

Linux, MacOs:

'''
ls
'''

Удаление файла в Windows: 

'''

del <filename>
'''

в Linux, MacOs:

'''
rm <filename>
'''
